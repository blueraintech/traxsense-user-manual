To create each specific task type, follow the instructions below:

**Add Item Inspection**

1. Fill in the following details:
   - Priority
   - Users
   - Note
   - Item Type Filters
   - Item Status Filters
   - Locations
   - Create task per location
   - Group inventory by
   - RFID Reader Power
2. Click "Save" when done.
