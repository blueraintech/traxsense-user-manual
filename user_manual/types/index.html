    Types

    The Type Management section of the system provides administrators with web-based interfaces to easily create, edit, and manage different statuses, types, and configurations of items, transactions, and devices within the system.

    Item Status Management

    View Item Status

    To view a list of Item Status types, follow these steps:

    1. Click on the Item Status option within Types.
    2. You will be redirected to the Item Status Management page, where you can view the item types.

    Add Item Status

    To add a new item status type, follow these steps:

    1. On the Item Status Management page, click the "Add" button.
    2. Fill in the required fields.
    3. Click the "Save" button to create the new item status.

    Edit Item Status

    To edit an existing item status type, follow these steps:

    1. On the Item Status Management page, locate the item status you want to edit.
    2. Click Status Id of the status you wish to edit.
    3. You will be redirected to the Edit Item Status page.
    4. Modify the desired fields.
    5. Click the "Save" button to save the changes.

    Delete Item Status

    To delete an existing item status type, follow these steps:

    1. On the Item Status Management page, locate the item status you want to edit.
    2. Click Status Id of the status you wish to edit.
    3. You will be redirected to the Edit Item Status page.
    4. Click on the "Delete" button next to the item status.
    5. A warning pop-up will appear.
    6. Confirm the deletion by clicking the "Confirm" button.

    Item Type Management

    View Item Types

    To view a list of Item Types, follow these steps:

    1. Click on the Item Types option within Types.
    2. You will be redirected to the Item Status Management page, where you can view the item types.

    Import Item Types

    The Item Type importer allows for the bulk adding of multiple Item Types from a csv. You need to place all these Item Types within a single organization.

    Add Item Type

    To add a new item type, follow these steps:

    1. On the Item Type Management page, click the "Add" button.
    2. Provide the following details for the new item type:
    3. Item Type Name: Enter the name of the item type.
    4. Item Type Description: Add a description for the item type.
    5. Item Type Barcode: Specify the barcode for the item type.
    6. Organization: Select the organization associated with the item type.
    7. Optional Condition Types: Add or modify any optional condition types.
    8. Optional Item Image: Attach an optional image for the item type.
    9. Use an existing template or Add Metadata: Choose either to use an existing template or add metadata.

    If using an existing template:

    • Select the desired item type template.

    If adding metadata:

    • Provide the Field Name, Data Type, and Default Value.

    • Click the "Submit" button to create the new item type.

    Edit Item Type

    To edit an existing item type, follow these steps:

    1. On the Item Type Management page, locate the item type you want to edit.
    2. Click Type Id of the type you wish to edit.
    3. You will be redirected to the Edit Item Type page.

    4. Modify the desired fields:

    5. Item Type Name: Change the name of the item type.

    6. Item Type Description: Update the description of the item type.
    7. Item Type Barcode: Modify the barcode associated with the item type.
    8. Organization: Change the organization associated with the item type.
    9. Optional Condition Types: Add or modify any optional condition types.
    10. Optional Item Image: Attach a new or updated image for the item type.

    11. Use an existing template or Add Metadata:

      • If using an existing template:
      • Select a different item type template if desired.
      • If adding metadata:
      • Change the Field Name, Data Type, and Default Value as needed.

    12. Click the "Submit" button to save the changes.

    Delete Item Type

    To delete an existing item type, follow these steps:

    1. On the Item Type Management page, locate the item type you want to delete.
    2. Click Type Id of the type you wish to delete.
    3. You will be redirected to the Edit Item Type page.
    4. Click on the "Delete" button in the item type form.
    5. A warning pop-up will appear. You can optionally also delete the Item Type template associated with the Item Type.
    6. Confirm the deletion by clicking the "Confirm" button.
